# 🗓️ 위클리

## 🥡 의존성 설치하기

```shell
yarn
```

## 💻 개발하기

- 환경변수 예시 복사 후 설정

```shell
cp .env.example .env
```

- 개발서버 켜기

```shell
yarn dev
```

- 개발서버 포트 정보

| App | Port |
| :-: | :-: |
| attendance | `:3000` |
| admin | `:3001` |
| qr | `:3002` |
